# 📊 Sentiment Analyzer Pro

An **AI-powered sentiment and emotion analysis web application** built with **Python and Streamlit**.
It analyzes text, PDFs, or Word documents to detect **sentiment polarity and emotional tone** using **AFINN-165** and the **NRC Emotion Lexicon**.

# 🚀 Features

## 📄 Document Analysis

Upload **PDF or Word (.docx)** documents and automatically extract text for sentiment analysis.

## ✏️ Text Analysis

Paste any text such as:

* Reviews
* Emails
* Feedback
* Articles

and receive instant analysis.

## 😀 Sentiment Detection

Uses the **AFINN-165 lexicon** to classify text into:

* Positive
* Negative
* Neutral

with **log-normalized scoring**.

## 🧠 Emotion Detection

Identifies **8 core emotions** using the NRC Emotion Lexicon:

* Joy
* Sadness
* Anger
* Fear
* Disgust
* Surprise
* Trust
* Anticipation

## 📊 Interactive Visualizations

Includes:

* Emotion **radar charts**
* Sentiment score cards
* Emotion distribution bars

powered by **Plotly** and **Pandas**.

## 🔍 Word-Level Insights

Automatically detects and highlights:

* ✅ Positive words
* ⚠️ Negative words

within the analyzed text.

## 📈 Sentence-Level Analysis

Displays a sequence chart showing how sentiment evolves sentence by sentence throughout your document.

## 📥 Detailed Reports

Download a plain text summary of your sentiment score, word counts, and emotion breakdown directly from the app.

## 🔐 Authentication System

Secure login system with session-based access.

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Hemanthram0205/Sentiment-Analysis.git
```

Move into the project folder:

```bash
cd Sentiment-Analysis
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
